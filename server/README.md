# Aero Server - Backend


## How do i run this locally?

### Environment
You must include the following in `/server/.env`
```
PORT = PORT (ex: 3000)
DB_HOST = MongoDb
```

### Terminal
Open the terminal and run the following from `/`
```
cd server
npm install
npm run tsc
npm start
```
You can also run
```
cd server
npm install
npm run develop
```
